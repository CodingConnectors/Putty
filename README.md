# Putty

## 프로젝트 소개 
- 디자인은 애견샵 퍼피독을 참고하고
- 기능은 컴퓨존을 참고하여 만든 쇼핑몰입니다.

## 개발 기간
23.11.24 ~

## 멤버구성
- 팀장 조영훈
- 부팀장 최현종
- Time-Keeper 최현종
- 서기 김병욱

## 개발환경
- `java 17`
- `JDK 17.0.9`
- IDE : STS 4.20.0
- Framework : Springboot(2.7.17)
- Database : amazon RDS MariaDB 10.6.14
- ORM : JPA
- Websocket(java 17이상), STOMP, rebbitMQ
- Tamplate : thymeleaf
              lombok
